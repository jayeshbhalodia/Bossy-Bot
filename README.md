# Bossy-bot

Bot that will keep you motivated!

If you are interested to support on this App then contact me : jsbhalodia.dev@gmail.com


## Installtion & Start
- npm i

## Start it on local
- npm run dev
- npm run build
- npm start

## Build
- npm run build
- npm run pack ( Mac OS)
- npm run pack-linux ( Ubuntu / Debian )
